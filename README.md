# thefirstcaptain.github.io

Personal GitHub Pages site for `gartside.net`, currently serving a simple Linktree-style landing page.

## Page Overview

The homepage (`index.html`) presents:
- A profile image (`images/profile.jpg`)
- Title text: "Kirk's Corner of the Internet"
- Short description: "Links to my family tree and socials"
- A list of external links, including:
  - Mac Family Tree
  - Reddit
  - Bluesky
  - Mastodon
  - Substack
  - LinkedIn
  - Facebook
  - Instagram
  - X

## Files

- `index.html`: Page structure and outbound links
- `style.css`: Visual styling for the page
- `images/`: Image assets used by the page
- `CNAME`: Custom domain configuration for GitHub Pages

## Notes

- Font Awesome icons are loaded via CDN in `index.html`.
- The page is static and does not require a build step.
